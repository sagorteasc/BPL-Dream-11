
# 🏏 BPL Dream 11

Build your dream team, manage your coins, and select the best players with a clean and responsive interface.

---

## ✨ Features

- 🪙 Coin system to claim and manage coins
- 🏏 Select players to build your dream team
- ❌ Remove selected players anytime
- 🔔 Toast notifications for user actions
- 📱 Fully responsive design
- ⚡ Fast and smooth user experience

---

## 🛠 Tech Stack

- **React.js**
- **Vite**
- **Tailwind CSS**
- **JavaScript (ES6)**
- **React Toastify**
